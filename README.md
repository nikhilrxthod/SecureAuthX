# SecureAuthX
SecuAuthX is a secure Flask-based authentication system featuring user registration, login protection, password hashing, brute-force mitigation, account lock mechanisms, cooldown rounds, IP-based login tracking, and login attempt monitoring using MySQL for backend data management and security auditing.


1.	create database using commands in "database.txt"
2.	pip install -r requirements.txt
3. goto \config -> open database_config.py & email_config.py -> enter your credentials
4. run app.py -> link dega usko open krna bas
